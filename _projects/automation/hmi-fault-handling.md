## Project Overview

Redesigned HMI screens and fault-handling routines to improve operator usability and reduce downtime in automated production lines.

### Objectives
- Improve operator workflow
- Reduce unplanned stoppages
- Standardize HMI alarm handling

### Scope of Work
- HMI screen redesign and layout improvements
- Alarm categorization and logging
- Integration with PLC for fault detection
- Operator training and documentation

### Tools & Technologies
- Pro-face HMI
- CLICK PLCs
- Ladder logic & scripting
- Data logging

### Results
- Faster fault resolution
- Reduced operator error
- Improved reporting for maintenance
